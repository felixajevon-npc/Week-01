## A. Ringkasan Praktikum
Pada pertemuan pertama ini, fokus utama adalah persiapan lingkungan pengembangan (*development environment*) untuk Flutter. Saya telah berhasil melakukan instalasi dan konfigurasi perangkat lunak yang dibutuhkan:

1.  **Instalasi Flutter SDK:** Mengunduh dan mengatur *environment variables* agar perintah `flutter` dapat dijalankan dari terminal.
2.  **Konfigurasi Android Studio:** Menginstal Android SDK Command-line Tools dan menyetujui lisensi Android (mengatasi masalah *licenses not accepted*).
3.  **Setup Emulator:** Membuat *Virtual Device* (Pixel 7 API 36) untuk pengujian aplikasi.
4.  **Uji Coba Hello World:** Berhasil menjalankan aplikasi pertama ("Aplikasi Felix") pada emulator tanpa error.

## B. Deskripsi Aplikasi (Hello World)
Pada praktikum ini, saya telah membuat aplikasi sederhana menggunakan widget `MaterialApp` dan `Scaffold` sebagai struktur utamanya. Berikut adalah rincian tampilan visual aplikasi yang telah dibuat:

* **App Bar (Bagian Atas):**
    Aplikasi ini memiliki *header* dengan judul teks **"Aplikasi Felix"**. Latar belakang *App Bar* diatur berwarna **Putih** (`Colors.white`) untuk memberikan tampilan yang bersih.

* **Body (Bagian Isi):**
    Bagian utama layar menampilkan teks **"Hello World"**. Teks ini diatur menggunakan widget `Center` sehingga posisinya berada tepat di tengah-tengah layar (secara vertikal maupun horizontal).

* **Styling Teks:**
    Teks "Hello World" tersebut diberi gaya (*style*) dengan ukuran font **30**, sehingga terlihat lebih besar dan jelas dibandingkan teks standar.
